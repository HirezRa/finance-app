# Changelog

## [1.0.0] - 2026-04-05

### Added

- ניהול חשבונות בנק וכרטיסי אשראי (מוסדות ישראליים נתמכים בסקרייפר)
- סנכרון אוטומטי עם `israeli-bank-scrapers` (fork מתועד בפרויקט)
- סיווג אוטומטי לפי מילות מפתח בקטגוריות
- תקציב חודשי עם מעקב בזמן אמת מול הוצאה בפועל
- לוח בקרה עם ניווט חודשים, גרף שבועי ופילוח קטגוריות
- הגדרות משכורת (`effectiveDate`, טווח ימים)
- זיהוי עסקאות Pending והתאמה ל־Completed (מניעת כפילות בשדרוג סטטוס)
- מניעת כפילות חיובי אשראי (החרגה מחשבון בנק מול עסקאות כרטיס)
- הערות לעסקה (`note`) וסימון "לא בתקציב"
- כינוי ותיאור לחשבונות
- התראות תקציב (אזהרה / חריגה) וסף הוצאה גדולה
- אינטגרציה עם OLLAMA לסיווג (אופציונלי)
- אינטגרציה עם n8n להתראות (webhook)
- אבטחה: JWT, Refresh, הצפנת אישורי בנק, 2FA (TOTP), Rate Limiting

### Documentation

- תיעוד שכבת `docs/`: תכונות, מסד נתונים, משכורת, חיובי אשראי, pending, פריסה, פתרון בעיות
